TimeAndSales (C#)
===============================================================================

Overview
-------------------------------------------------------------------------------

This example demonstrates using the TT API to retrieve time & sales data from a 
single instrument.


Instructions
-------------------------------------------------------------------------------

1. Drag and drop an Instrument from the Market Grid in X_TRADER.


TT API Objects
-------------------------------------------------------------------------------

TimeAndSalesSubscription
