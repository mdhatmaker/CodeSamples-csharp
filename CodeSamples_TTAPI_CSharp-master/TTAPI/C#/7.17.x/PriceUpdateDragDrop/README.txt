PriceUpdateDragDrop (C#)
===============================================================================

Overview
-------------------------------------------------------------------------------

This example demonstrates using the TT API to retrieve market data from a 
single instrument.  


Instructions
-------------------------------------------------------------------------------

1. Drag and drop an Instrument from the Market Grid in X_TRADER.
2. View price updates.


TT API Objects
-------------------------------------------------------------------------------

PriceSubscription
