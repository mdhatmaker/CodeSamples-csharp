PriceUpdateDepth (C#)
===============================================================================

Overview
-------------------------------------------------------------------------------

This example demonstrates using the TT API to retrieve market depth data from a 
single instrument. 

Note: The maximum level of depth can differ by exchange.


Instructions
-------------------------------------------------------------------------------

1. Drag and drop an Instrument from the Market Grid in X_TRADER.
2. View market depth


TT API Objects
-------------------------------------------------------------------------------

PriceSubscription
